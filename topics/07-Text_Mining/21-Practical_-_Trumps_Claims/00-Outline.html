Classifying Trump's false claims while in office

Outline

This week's practical (and assignment) is based on the dataset compiled by the Washington Post on the false claims made by Trump while in office. Of the 30,573 claims, 30,404 have been categorised using 18 categories (Immigration (3,225 claims), Foreign policy, Election, ..., Terrorism (72 claims)) also each claim has been dated, tagged as to location (Campaign rally, Remarks, Interview, Twitter (of course), etc.), rationale for claim's falsehood, and is cross linked to related claims.

The aim of this practical is to build a classifier — using only columns claim, location, date and target category — that predicts the claim category. Given that this is a multi-class (18) problem, and unbalanced, but we can assume that cost of Type I vs Type II errors the same, we are going to use the macro-F1 metric.

In the practical we will go over the construction of the basic model and for the assignment I want you to improve on this basic model.
